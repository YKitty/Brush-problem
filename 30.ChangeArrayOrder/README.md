题目：调整数组的顺序让奇数位于偶数的前面        
思路：      
  采用左右指针的方法      
  left指针和right指针         
  当left <= right的时候              
  从左边开始找，让left指针指向偶数           
  从右边开始找，让right指针指向奇数        
  然后交换两个指针的数据即可        
  结束条件：left > right          
           

扩展：
  让三的倍数的数放在不是三的倍数的数值前面            
  所以在判断数值的时候，可以写一个函数来进行判断，这样就可以做到对于一类问题进行求解
